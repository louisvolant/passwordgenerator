# passwordgenerator

Generate apple-style random passwords

## How to execute

Go in the same folder than the python script.

1. Default run of the script (10 random passwords generated, 20 characters total)

````
$ python3 passwordgenerator.py
````

2. Run the script with 15 random passwords generated (20 characters total)

````
$ python3 passwordgenerator.py 15
````

2. Run the script with 10 random passwords generated & 20 characters total

````
$ python3 passwordgenerator.py 10 30
````
